# HR Analytics Dashboard | Power BI Project

## Overview

This project is an interactive HR Analytics Dashboard developed in Microsoft Power BI to analyze employee workforce data and identify key HR insights such as employee attrition, salary trends, department performance, age distribution, and overtime impact.

The dashboard helps HR teams and business managers monitor workforce metrics and make data-driven decisions to improve employee retention and organizational performance.

---

## Key Features

* Employee Overview KPIs
* Attrition Analysis
* Department-wise Employee Distribution
* Gender Distribution Analysis
* Salary Insights by Job Role
* Overtime vs Attrition Analysis
* Interactive Slicers and Filters
* Professional Dashboard UI Design
* Multi-page Navigation Sidebar

---

## KPIs Included

* Total Employees
* Active Employees
* Employees Left
* Attrition Rate
* Average Salary
* Average Employee Age

## Dashboard Insights

* Research & Development department has the highest attrition.
* Employees aged 26–35 form the largest workforce group.
* Overtime employees show comparatively higher attrition.
* Salary distribution varies significantly across job roles.
